title: i64 Instructions

## i64.add

Add two i64 values.

**Signature:** `(param i64 i64) (result i64)`

**Example:**
```wat
(i64.add (i64.const 5000000000) (i64.const 3000000000))
```

## i64.const

Create a constant i64 value.

**Signature:** `(result i64)`

**Example:**
```wat
(i64.const 9223372036854775807)  ;; Max i64
(i64.const -1)
```
